# 唐宋诗人星空可视化页面

## 概述

「唐宋诗人星空」是一个交互式的三维可视化项目，将唐朝诗人、宋朝词人、唐朝将星和宋朝将星以星空的形式呈现。每个文人化作一颗星星，星星的大小和亮度反映其名气和成就，星星之间的连线代表人物之间的关系。

现在支持**唐朝诗人**、**宋朝词人**、**唐朝将星**和**宋朝将星**四个独立的星空场景，可以自由切换。

## 访问方式

直接在浏览器中打开 `tang-song-poets-stars.html` 文件即可体验。

## 功能特性

### 1. 四类星空

| 星空 | 人数 | 时期 | 星星颜色 |
|------|------|------|----------|
| 唐朝诗人星空 | ~150位 | 初唐、盛唐、中唐、晚唐 | 金色系 |
| 宋朝词人星空 | ~50位 | 北宋、南宋 | 蓝色系 |
| 唐朝将星星空 | ~55位 | 唐朝 | 红色系 |
| 宋朝将星星空 | ~20位 | 宋朝 | 紫色系 |

### 2. 一键切换

点击顶部标签页即可在不同星空之间切换：
- 唐朝星空
- 宋朝星空
- 唐朝将星
- 宋朝将星

### 3. 分类筛选

左侧筛选面板支持：
- **诗人**（默认勾选）：控制诗人星空的显示/隐藏
- **将星**（默认勾选）：控制将星星空的显示/隐藏

### 4. 名气分级

- 🌟 **一级（最亮最大）**：诗仙/诗圣/词圣/千古第一才女/军神等顶级人物
- 🌟 **二级**：名家级别的诗人/词人/名将
- 🌟 **三级**：重要诗人/著名将领
- 🌟 **四级（较小较暗）**：一般人物

### 5. 关系网络

- **挚友/亲属/师徒/君臣**：明亮连线
- **文坛同事/战友**：中等亮度连线
- **泛泛之交/崇敬**：暗淡连线

### 6. 交互操作

| 操作 | 说明 |
|------|------|
| 拖拽 | 旋转视角，查看不同方向的星星 |
| 滚轮 | 缩放视角距离 |
| 点击 | 查看人物详细信息 |
| 关系人名 | 点击可跳转到对应人物 |

### 7. 信息面板

点击任意星星后，右侧会显示：
- 姓名
- 所属时代
- 成就标签
- 代表作品（可点击查看全文）
- 人物关系网络

### 8. 跨分类关系

- 唐朝将星可与唐朝诗人建立关系（如郭子仪与李白）
- 宋朝将星可与宋朝词人建立关系（如岳飞与辛弃疾）
- 关系跳转时会自动切换到对应的星空场景（仅限同类目内跳转）

## 技术实现

- **Three.js**：用于3D渲染
- **纯HTML/CSS/JS**：无需构建工具，直接浏览器运行
