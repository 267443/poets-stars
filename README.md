# 唐宋诗人星空可视化页面

## 概述

「唐宋诗人星空」是一个交互式的三维可视化项目，将唐朝诗人和宋朝词人以星空的形式呈现。每个文人化作一颗星星，星星的大小和亮度反映其名气和成就，星星之间的连线代表人物之间的关系。

现在支持**唐朝**和**宋朝**两个独立的星空场景，可以自由切换。

## 访问方式

直接在浏览器中打开 `tang-poets-stars.html` 文件即可体验。

## 功能特性

### 1. 双朝代星空

- **唐朝星空**：150位诗人，涵盖初唐、盛唐、中唐、晚唐四个时期（金色系星星）
- **宋朝星空**：50位词人，分为北宋、南宋两个时期（蓝色系星星）
- **一键切换**：点击顶部「唐朝星空」或「宋朝星空」标签即可切换

### 2. 名气分级

- 🌟 **一级（最亮最大）**：诗仙/诗圣/词圣/千古第一才女等顶级文人
- 🌟 **二级**：名家级别的诗人/词人
- 🌟 **三级**：重要诗人
- 🌟 **四级（较小较暗）**：一般诗人

### 3. 关系网络

- **挚友/亲属/师徒**：明亮连线
- **文坛同事**：中等亮度连线
- **泛泛之交/崇敬**：暗淡连线
- **没有关系**：无线连接

### 4. 交互操作

| 操作 | 说明 |
|------|------|
| 拖拽 | 旋转视角，查看不同方向的星星 |
| 滚轮 | 缩放视角距离 |
| 点击 | 查看诗人/词人详细信息 |
| 关系人名 | 点击可跳转到对应人物 |

### 5. 信息面板

点击任意星星后，右侧会显示：
- 姓名
- 所属时代
- 成就标签（诗仙、诗圣、文豪、词中之龙等）
- 代表作品（可点击查看全文）
- 人物关系网络

### 6. 交互效果

- **星星脉动**：知名度越高，脉动越快越明显
- **点击反馈**：点击星星时触发闪烁动画
- **关系线高亮**：选中人物时，高亮显示其关系网络
- **跨朝代导航**：点击关系人时，如果在不同朝代会自动切换

## 数据规模

| 朝代 | 人数 | 时期 |
|------|------|------|
| 唐朝 | 150位 | 初唐、盛唐、中唐、晚唐 |
| 宋朝 | 50位 | 北宋、南宋 |

## 技术实现

- **Three.js**：用于3D渲染
- **纯HTML/CSS/JS**：无需构建工具，直接浏览器运行
