# 唐朝诗人星空可视化页面

## 概述

「唐朝诗人星空」是一个交互式的三维可视化项目，将52位唐代诗人以星空的形式呈现。每位诗人化作一颗星星，星星的大小和亮度反映诗人的名气和成就，星星之间的连线代表诗人之间的关系，线条的明暗程度表示关系的深浅。

## 访问方式

直接在浏览器中打开 `tang-poets-stars.html` 文件即可体验。

## 功能特性

### 1. 星星可视化

- **52位唐朝诗人**：涵盖初唐、盛唐、中唐、晚唐四个时期
- **名气和成就分级**：
  - 🌟 **一级（最亮最大）**：李白、杜甫、王维、白居易、杜牧、李商隐
  - 🌟 **二级**：孟浩然、高适、岑参、韩愈、柳宗元、刘禹锡、元稹、李贺、温庭筠、韦庄
  - 🌟 **三级**：其他重要诗人
  - 🌟 **四级（较小较暗）**：一般诗人

### 2. 关系网络

- **挚友/亲属**：明亮连线
- **文坛同事**：中等亮度连线
- **泛泛之交**：暗淡连线
- **没有关系**：无线连接

### 3. 交互操作

| 操作 | 说明 |
|------|------|
| 拖拽 | 旋转视角，查看不同方向的星星 |
| 滚轮 | 缩放视角距离 |
| 点击 | 查看诗人详细信息 |

### 4. 信息面板

点击任意星星后，右侧会显示：
- 诗人姓名
- 所属时代（初唐/盛唐/中唐/晚唐）
- 代表作品
- 人物关系网络

## 诗人数据

### 初唐时期（618年-712年）

王绩、卢照邻、骆宾王、王勃、杨炯、宋之问、贺知章、张若虚、陈子昂、杜审言、张九龄、王翰

### 盛唐时期（713年-755年）

王之涣、孟浩然、王昌龄、李白、王维、杜甫、高适、岑参、崔颢、王湾、张旭、王冰

### 中唐时期（756年-835年）

韩愈、柳宗元、刘禹锡、白居易、元稹、贾岛、李贺、孟郊、张籍、王建、刘方平、顾况、卢仝、李益

### 晚唐时期（836年-907年）

杜牧、李商隐、温庭筠、韦庄、罗隐、杜荀鹤、郑谷、鱼玄机、韩偓、李群玉、杜秋娘、聂夷中、秦韬玉、来鹏

## 扩展说明

如需修改诗人数据或关系网络，可编辑 `tang-poets-stars.html` 文件中的 `poetsData` 数组：

```javascript
{
    id: 1,
    name: "诗人姓名",
    era: "所属时代",
    works: ["代表作1", "代表作2"],
    fame: 1-4,  // 名气等级
    relations: [
        { name: "关系人", type: "关系类型", strength: "strong/medium/weak" }
    ]
}
```
